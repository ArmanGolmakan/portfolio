# Personal Portfolio — Arman Golmakan

My personal portfolio site built with Next.js, TypeScript, and Tailwind CSS. Designed with a focus on clean UI, smooth interactions, and fast page loads.

**Live site:** [armangolmakan.com](https://armandev.me)

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Runtime:** React 19

## Features

- Single-page layout with sticky sidebar navigation
- Active section tracking via IntersectionObserver
- Responsive design — mobile-first with a two-column desktop layout
- Experience, Projects, and About sections

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── app/             # Next.js app router entry
├── components/      # Shared components (Sidebar, etc.)
└── sections/        # Page sections (About, Experience, Projects, Footer)
```
